# Power BI Interactive Dashboard

## 📌 Project Overview
This project demonstrates the creation of an interactive sales dashboard using Microsoft Power BI. It focuses on data exploration through interactive visualizations, filters, slicers, drill-down, drill-through, and DAX measures to provide meaningful business insights.

## 📊 Dashboard Features
- Interactive Sales Dashboard
- Slicers for dynamic filtering
- Visual, Page, and Report-Level Filters
- Drill Down using hierarchies
- Drill Through for detailed analysis
- DAX Measure-Based Filtering
- Sales Trend Analysis
- Product-wise Sales Distribution

## 🛠️ Tools & Technologies
- Microsoft Power BI
- DAX (Data Analysis Expressions)
- Microsoft Excel
